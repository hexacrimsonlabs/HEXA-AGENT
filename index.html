⟳ Installation

Deploy in seconds

01

Download

Run the one-liner installer from HEXACRIMSON LABS. It auto-detects your OS and architecture, downloads the signed binary, and verifies the checksum.

curl -sSL https://hexacrimson.io | sh
02

Authenticate

The Hexacrimson agent generates a unique identity and registers with your gateway using a short-lived enrollment token.

agent enroll --token hxl-7f3a...e9b2
03

Manage

Once connected, manage your Hexacrimson agent from the dashboard — push configs, run commands, view metrics in real time.

agent status   # → active (running)
